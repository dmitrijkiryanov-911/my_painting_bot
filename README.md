\# my\_painting\_bot



Telegram‑бот для учёта картин, сроков хранения и автоматических напоминаний.



\## Возможности



\- Добавление картины с датой передачи и сроком хранения

\- Автоматический расчёт даты забора

\- Просмотр всех своих заказов

\- Экспорт заказов в Excel

\- Ежедневные напоминания:

&nbsp; - за 7 дней до даты забора

&nbsp; - за 2 дня

&nbsp; - в день забора



\## Стек технологий



\- Python 3

\- Aiogram 3

\- OpenPyXL

\- JSON‑база данных



\## Структура проекта



my\_painting\_bot/

│

├── reminder.py        # ежедневные напоминания

├── requirements.txt   # зависимости

└── README.md





\## Как запустить локально



1\. Установить зависимости: pip install -r requirements.txt

2\. Создать переменную окружения:export TOKEN="твойтокенбота"

3\. Запустить бота:python bot.py



\## Деплой на Render



1\. Создать репозиторий на GitHub и загрузить файлы

2\. На Render создать Web Service → выбрать репозиторий

3\. Build Command: pip install -r requirements.txt

4\. Start Command:python bot.py

5\. В разделе Environment добавить переменную:TOKEN = <твой токен>



\## Напоминания



Для ежедневных уведомлений можно использовать Render Cron Jobs или внешний cron‑сервис:python reminder.py
